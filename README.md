Ключевые принципы:
- Единая шкала мастерства: Объяснение → Умение → Навык → Мастерство.
- Мостик к грейдам: Junior ≈ Объяснение/Умение; Middle ≈ Навык; Senior/Lead ≈ Мастерство.
- Четыре оси карьерного роста: профессиональный рост, должностной рост, калибр личности, траектория ролей.
- Инкрементальный подход к контракту: v1.0 (Неделя 1) → v4.0 (Неделя 8).

## 🚀 Инструменты для работы с курсом

### Course CLI — Работа через командную строку

**Быстрый старт:**
```bash
# Установка
cd course_cli
pip install -e .
cd ..

# Основные команды
course start-week 1          # Начать неделю 1
course contract init         # Создать личный контракт
course progress              # Показать прогресс
course template --list       # Список шаблонов
course info                  # Информация о курсе
```

**Что даёт:**
- ✅ Быстрая навигация по неделям через чат в Cursor/IDE
- ✅ Автоматическое создание контрактов из шаблонов
- ✅ Отслеживание прогресса по курсу
- ✅ Удобная работа с шаблонами

**Документация:** `course_cli/README.md`

### 🎯 Настройка Cursor для автоматической работы

**Для полной интеграции с Cursor:**
1. Установите CLI (см. выше)
2. Откройте проект в Cursor
3. Cursor автоматически подхватит настройки из `.cursorrules`

**Теперь можете работать через чат:**
- "Начни неделю 1" → автоматически откроет материалы
- "Создай контракт" → создаст файл из шаблона
- "Покажи прогресс" → выведет таблицу прогресса

**Подробная инструкция:** `CURSOR_SETUP.md`

### 🔧 Для разработчиков и поддержки проекта

**Система автоматического контроля качества**

Мы настроили систему, которая автоматически проверяет, что вся документация курса актуальна и корректна. Это означает, что участники всегда получают самую свежую и точную информацию.

**Как это работает:**
- 📋 **Автоматическая проверка** — система сама следит за обновлениями
- 🔍 **Контроль качества** — проверяет, что все ссылки работают
- ⚡ **Мгновенные обновления** — изменения сразу отражаются в документации

**Технические детали:** см. `scripts/README.md`

## Три способа работы

### 1. Традиционный (как раньше)
- Открываете файлы недель напрямую
- Редактируете контракты в любом редакторе
- Используете Excel/Notion для таблиц

### 2. Через CLI (командная строка)
- Работаете через команды `course`
- Быстрая навигация и создание файлов
- Отслеживание прогресса

### 3. Через Cursor (с AI-ассистентом)
- Работаете через чат: "Начни неделю 1"
- Cursor автоматически выполняет CLI команды
- Полная интеграция с контекстом курса

**Выбирайте удобный для вас способ! Все три варианта равноценны.**
