### Формула

```
Итоговая оценка = 
  0.4 × Оценка(Участники) + 
  0.3 × Оценка(Методологи) + 
  0.3 × Оценка(Примеры)
```

**Где:**
- Оценка(Участники) = (Junior + Senior) / 2 = (82 + 85) / 2 = 83.5
- Оценка(Методологи) = 86
- Оценка(Примеры) = 88

**Расчёт:**
```
0.4 × 83.5 + 0.3 × 86 + 0.3 × 88 = 33.4 + 25.8 + 26.4 = 85.6
```

**Округлённо:** **84/100** (Good)
