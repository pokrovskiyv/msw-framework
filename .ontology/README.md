# Systemic Career v2.3

Онтология проекта, управляемая через Ontology Toolkit.

## Структура

- `concepts/` — понятия (C_*)
- `methods/` — методы (M_*)
- `systems/` — системы (S_*)
- `problems/` — проблемы (P_*)
- `artifacts/` — артефакты (A_*)

## Использование

```bash
# Добавить понятие
ontology add "Название понятия"

# Посмотреть список
ontology list

# Проверить онтологию
ontology audit

# Экспортировать
ontology export --format csv
```
